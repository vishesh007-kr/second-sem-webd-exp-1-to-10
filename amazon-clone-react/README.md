# Amazon Clone (React + Vite)

A minimal React + Vite implementation of the responsive Amazon-like demo with working cart persisted to localStorage.

Run locally:

```bash
cd amazon-clone-react
npm install
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

What's included:
- `src/App.jsx` main app
- `src/components/ProductCard.jsx` product UI
- `src/components/Cart.jsx` cart panel with quantity controls
- `localStorage` persistence
